# RecipeGenerator

A web application that generates recipes based on ingredients you have on hand. Built during **Grizzhacks 2024** hackathon.

> **Note:** This repository is a fork of [benpentecost2004/RecipeGenerator](https://github.com/benpentecost2004/RecipeGenerator). I contributed to this project during Grizzhacks 2024 alongside @benpentecost2004.

## Features

- Input available ingredients
- Generate matching recipes
- Simple web interface
- Command-line version also available

## Technologies Used

- Python
- Flask
- HTML5
- CSS3

## Installation

```bash
git clone https://github.com/vmgriest/RecipeGenerator.git
cd RecipeGenerator
pip install -r requirements.txt
python app.py
